# Game-Developement-Practice-Project-Walkthrough
This simple JavaScript game follows a walkthrough that was developed by Franks Laboratory, YouTube

## Technologies
+ HTML
+ CSS
+ JavaScript

## Features
+ Infinite scroll parallax background
+ Scoreboard System with build in Game-Over mechanics
+ Various enemy types with hit-boxes
+ Simply intuative design

## Controls
+ Arrow Keys for movement
+ Space bar for jumping
+ Hold enter to engage in 'attack mode'

## The Process
The project you see here was created using a walkthrough series of smaller game design projects, 9 in total, that I have followed using the YouTube content creator Franks Laboratory. This was a long process of learning new concepts to me such as HTML canvas, to create the game world, animations in JavaScript wth sprite images, parallax background motion for realistic 3D effects in motion. I have learnt a lot from this series of videos, and will use this knowlege on future independent projects.
